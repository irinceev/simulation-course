## 1. Моделирование дискретной случайной величины

### Параметры эксперимента

- **Значения:** $X = \{1,\ 2,\ 3,\ 4,\ 5\}$
- **Вероятности:** $P(X = i) = 0.2$ для всех $i$ (равномерное дискретное)
- **Теоретическое среднее:** $E[X] = \sum x_i \cdot p_i = 3.0$
- **Теоретическая дисперсия:** $D[X] = \sum x_i^2 \cdot p_i - (E[X])^2 = 2.0$

### Алгоритм генерации

Алгоритм на каждом шаге вычитает вероятность $P_k$ из убывающей суммы $A$, начиная с $A = 1$, и возвращает значение $x_k$, как только выполняется условие $A \leq \alpha$, где $\alpha \sim U(0, 1)$ — базовый датчик равномерно распределённой случайной величины.

### Эмпирические вероятности

Эмпирическая вероятность $\hat{p}_i$ каждого значения $x_i$ вычисляется как:

$$\hat{p}_i = \frac{n_i}{N}$$

где $n_i$ — количество появлений значения $x_i$ в выборке, $N$ — объём выборки.

### Критерий хи-квадрат

Нулевая гипотеза $H_0$: эмпирическое распределение соответствует заданному теоретическому ряду. Статистика:

$$\chi^2 = \sum_{i=1}^{m} \frac{(n_i - N p_i)^2}{N p_i}$$

Гипотеза $H_0$ отклоняется, если $\chi^2 > \chi^2_\text{кр}(\alpha,\ m{-}1)$ при уровне значимости $\alpha = 0.05$.

При $N = 10$ случайные флуктуации велики, и критерий может отклонить гипотезу даже при корректной генерации. Начиная с $N = 100$ критерий стабильно подтверждает соответствие теории.

### Вывод по дискретной СВ

- Алгоритм инверсии CDF корректно воспроизводит заданный ряд распределения.
- При $N = 10$ результаты нестабильны: относительные погрешности среднего и дисперсии достигают 10–30%, критерий $\chi^2$ может не выполняться из-за малого числа наблюдений.
- При $N = 100$ качество оценок заметно улучшается, критерий $\chi^2$ начинает стабильно выполняться.
- При $N \geq 1000$ погрешности среднего и дисперсии не превышают нескольких процентов, а значение статистики $\chi^2$ оказывается значительно меньше критического.
- Полученные результаты экспериментально подтверждают **закон больших чисел**: при увеличении объёма выборки эмпирические характеристики сходятся к теоретическим.

## 2. Моделирование нормальной случайной величины

### Параметры эксперимента

- **Среднее:** $\mu = 0$
- **Стандартное отклонение:** $\sigma = 1$
- **Теоретическая дисперсия:** $\sigma^2 = 1$
- **Теоретическая плотность:** стандартная нормальная $\mathcal{N}(0,\ 1)$

### Алгоритм генерации 

Генерация выполнена методом **преобразования Бокса–Мюллера**. Из двух независимых равномерных СВ $\alpha_1, \alpha_2 \sim U(0, 1)$ вычисляется стандартная нормальная СВ:

$$Z = \sqrt{-2 \ln \alpha_1} \cdot \cos(2\pi \alpha_2)$$

Затем осуществляется переход к $\mathcal{N}(\mu,\ \sigma^2)$:

$$X = \sigma \cdot Z + \mu$$

### Оценка точности

Для каждого объёма выборки вычислялись:

- Выборочное среднее: $\hat{\mu} = \dfrac{1}{N}\sum_{j=1}^N x_j$
- Выборочная дисперсия: $\hat{\sigma}^2 = \dfrac{1}{N}\sum_{j=1}^N (x_j - \hat{\mu})^2$
- Относительная погрешность дисперсии: $\delta_D = \dfrac{|\hat{\sigma}^2 - \sigma^2|}{\sigma^2} \cdot 100\%$

### Анализ гистограмм

- **$N = 10$:** Гистограмма имеет грубый «зашумлённый» вид. Форма колокола не прослеживается, данных недостаточно для статистических выводов.
- **$N = 100$:** Контуры нормального распределения начинают проявляться, однако гистограмма ещё содержит заметные случайные отклонения от теоретической кривой.
- **$N = 1000$:** Гистограмма близка к теоретической плотности, форма колокола чётко выражена. Небольшие локальные отклонения в отдельных интервалах.
- **$N = 10000$:** Гистограмма практически совпадает с теоретической кривой плотности $\mathcal{N}(0,1)$. Случайные отклонения минимальны.

### Вывод по нормальной СВ

- Метод Бокса–Мюллера корректно воспроизводит нормальное распределение с заданными параметрами.
- При $N = 10$ оценки среднего и дисперсии крайне нестабильны, гистограмма не отражает форму нормального распределения.
- Начиная с $N = 1000$ визуальная форма гистограммы и численные оценки совпадают с теоретическими значениями с высокой точностью.
- Критерий $\chi^2$ подтверждает статистическое соответствие сгенерированных данных нормальному закону при $N \geq 100$.
- При $\mu = 0$ погрешность среднего оценивается в абсолютном виде, так как относительная погрешность в этом случае не имеет смысла.

## 3. Общий вывод

1. **Корректность реализации.** Оба алгоритма — инверсия CDF для дискретной СВ и преобразование Бокса–Мюллера для нормальной СВ — реализованы в строгом соответствии с теоретическими формулами из курса стохастического моделирования. Генерация случайных величин происходит без систематических отклонений от заданных законов распределения.

2. **Закон больших чисел.** Во всех экспериментах наблюдается устойчивая сходимость: с ростом объёма выборки $N$ эмпирические характеристики ($\hat{M}$, $\hat{D}$, $\hat{p}_i$) монотонно приближаются к соответствующим теоретическим значениям. Это является прямым экспериментальным подтверждением закона больших чисел.

3. Малые выборки ($N = 10$) дают ненадёжные оценки: погрешности велики, критерий $\chi^2$ может отклонять корректную гипотезу из-за случайных флуктуаций. При $N = 10\,000$ погрешности оценок снижаются на порядок по сравнению с $N = 10$, а критерий стабильно подтверждает соответствие теории.

4. **Критерий $\chi^2$ как инструмент проверки.** Применение критерия хи-квадрат позволяет формально, на заданном уровне значимости $\alpha = 0.05$, проверить соответствие эмпирического распределения теоретическому. Результаты подтвердили, что критерий чувствителен к объёму выборки: при $N = 10$ он нередко даёт ложное отклонение из-за малого числа наблюдений в ячейках.

5.  Для получения статистически значимых и визуально убедительных результатов при моделировании как дискретных, так и непрерывных случайных величин рекомендуется использовать объёмы выборки не менее $N = 1000$.
