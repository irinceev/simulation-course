	- построить гистограммы

# Отчёт по лабораторной работе: Имитационное моделирование

## 1. Дискретное равномерное распределение

**Параметры:**
* **Значения:** $X = \{1, 2, 3, 4\}$
* **Вероятности:** $P(X=i) = 0.25$ для всех $i$
* **Теоретическое среднее:** $E[X] = 2.5$
* **Теоретическая дисперсия:** $D[X] = 1.25$

**Результаты моделирования:**
* При маленьких объемах выборки ($N=10$) наблюдаются заметные отклонения среднего и дисперсии от теоретических значений.
* При увеличении объема выборки ($N=1000, 10000$) эмпирические среднее и дисперсия практически совпадают с теорией.
* Относительные погрешности стабильно уменьшаются с ростом объема выборки.
* Критерий $\chi^2$ показывает согласие распределения выборки с теоретическим. Для больших $N$ значение статистики меньше критического, что подтверждает соответствие.

**Вывод:**
* Моделирование дискретного равномерного распределения выполнено корректно.
* Точность статистических оценок растет пропорционально увеличению объема выборки.
* Гистограммы визуально подтверждают равномерность распределения вероятностей.

---

## 2. Нормальное распределение

**Параметры:**
* **Среднее:** $\mu = 0$
* **Стандартное отклонение:** $\sigma = 1$
* **Теоретическая плотность:** стандартная нормальная кривая (Гауссиана)

**Результаты моделирования:**
* Для малых выборок ($N=10$) гистограмма имеет "шумный" вид, распределение выглядит случайным и неравномерным.
* Для $N=1000$ и $N=10000$ гистограмма почти полностью совпадает с теоретической кривой плотности.
* С увеличением объема выборки эмпирическая плотность сглаживается и принимает классическую колоколообразную форму.

**Вывод:**
* Моделирование нормального распределения методом Бокса-Мюллера корректно.
* Визуальная проверка подтверждает выполнение **закона больших чисел**: чем больше выборка, тем точнее эмпирическая плотность соответствует теоретической функции распределения.

---

## 3. Общий вывод

1.  **Успешная реализация:** Дискретное и непрерывное распределения моделируются программно без отклонений от заданных законов.
2.  **Закон больших чисел:** Экспериментально подтверждено, что с увеличением объема выборки эмпирические характеристики стремятся к своим математическим ожиданиям.
3.  **Статистическая проверка:** Критерий $\chi^2$ математически доказывает согласие полученных данных с исходной теорией.
4.  **Точность:** Для нормального распределения визуализация плотности подтверждает высокую точность выбранного алгоритма генерации.
5.  **Инструментарий:** Разработанный графический интерфейс (GUI) позволяет наглядно анализировать влияние объема данных на точность статистического вывода.
