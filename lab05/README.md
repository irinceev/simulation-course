### Моделирование случайных событий

# **В приложении реализованы два метода генерации случайных событий:***

Равномерная выборка из конечного множества с равными вероятностями:
P(Aᵢ) = 1/n, где n — количество исходов.
Реализация: random.choice(["ДА", "НЕТ"])
P(ДА) = P(НЕТ) = 0.5

# **Алгоритм Моисеева**

A := 1; k := 1
while True:
    A := A - Pk 
    if A ≤ ξ → произошло событие Ak

где ξ ~ U[0,1) — равномерно распределённая случайная величина (random.random()).
